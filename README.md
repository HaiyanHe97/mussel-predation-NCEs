# Effects of parasitic infection on predation risk responses in blue mussels (*Mytilus edulis*)

This repository contains the data and R code used to assess how parasitic infection status modulates the behavioural and physiological responses of blue mussels (*Mytilus edulis*) to predation risk cues from native and invasive crab predators.

## Overview

Mussels (infected vs. uninfected) were exposed to one of four water-borne treatments (Control, and three predation risk cue treatments: CM, HT, HS) and assessed across three behavioural/physiological assays:

1. **Aggregation and movement behaviour** — collective aggregation, locomotion, and byssus thread production
2. **Valve gape behaviour** — valve opening (fraction open) and its temporal variability (coefficient of variation, CV)
3. **Heart rate** — cardiac activity (bpm), measured via the [`heartbeatr`](https://github.com/EB-tools/heartbeatr) package

For each assay, two complementary statistical models were fitted:
- **Model A**: time as a continuous variable, to assess overall temporal trends
- **Model B**: time as a categorical variable, to assess differences at specific time points

## Data

### `data/raw/`

| File / folder | Description | Used by |
|---|---|---|
| `corrected_coordinates/` | Lens-distortion-corrected X/Y coordinate tracking data (one file per arena replicate) | `aggregation_01_calc_net_distance.R` |
| `Results_by_ind.csv` | Per-individual total distance travelled (Traveled_Dist) | `aggregation_02_build_arena_model.R` |
| `movement_row_clean.csv` | Per-arena infection metadata (intensity, byssus, shell length) | `aggregation_02_build_arena_model.R` |
| `Michaelis-Menten.csv` | Aggregation proportion (%) recorded at 5-min intervals per arena | `aggregation_03_merge_movement.R` |
| `gape_raw/` | Raw multi-channel valve gape sensor recordings (one CSV per sensor unit per replicate) | `valve_gape_01_import_raw.R` |
| `gape_metadata.csv` | Metadata linking sensor channels to individual mussels | `valve_gape_01_import_raw.R` |
| `heart_rate_raw/` | Raw PULSE heart rate sensor recordings (one folder per batch) | `heart_rate_01_calc_heart_rate.R` |
| `heart_rate_06-07_raw.csv` | Experimental metadata for heart rate individuals | `heart_rate_02_merge_metadata.R` |

### `data/processed/`

Key derived datasets used directly as input for the statistical models:

| File | Description |
|---|---|
| `merged_movement.csv` | Arena-level aggregation and movement dataset (final, used in `aggregation_01_analysis.R`) |
| `gap_5min_activity.csv` | Mussel x 5-min interval valve gape dataset (final, used in `valve_gape_01_analysis.R`) |
| `gap_5min_CV.csv` | Mussel x 5-min interval valve gape CV dataset (final, used in `valve_gape_02_CV_analysis.R`) |
| `heartrate_merged_final.csv` | Mussel x 5-min interval heart rate dataset (final, used in `heart_rate_01_analysis.R`) |

Other files in this folder (e.g. `arena_model.csv`, `all_gape_with_mussel.csv`, `net_distance_ALL.csv`, `heart_rate_by_batch/`) are intermediate outputs generated by the processing scripts, kept for transparency of the full data-processing pipeline.

## Scripts

### `scripts/processing/` — raw data to processed datasets

| Script | Input | Output |
|---|---|---|
| `aggregation_01_calc_net_distance.R` | `corrected_coordinates/` | `net_distance_ALL.csv` |
| `aggregation_02_build_arena_model.R` | `Results_by_ind.csv`, `net_distance_ALL.csv`, `movement_row_clean.csv` | `arena_model.csv` |
| `aggregation_03_merge_movement.R` | `Michaelis-Menten.csv`, `arena_model.csv` | `merged_movement.csv` |
| `valve_gape_01_import_raw.R` | `gape_raw/`, `gape_metadata.csv` | `all_gape_with_mussel.csv` |
| `valve_gape_02_process_5min.R` | `all_gape_with_mussel.csv` | `gap_5min_activity.csv` |
| `valve_gape_03_process_CV.R` | `all_gape_with_mussel.csv` | `gap_5min_CV.csv` |
| `heart_rate_01_calc_heart_rate.R` | `heart_rate_raw/` | `heart_rate_by_batch/*_heartrate.csv` |
| `heart_rate_02_merge_metadata.R` | `heart_rate_by_batch/`, `heart_rate_06-07_raw.csv` | `heartrate_merged_final.csv` |

### `scripts/analysis/` — statistical models and figures

| Script | Description |
|---|---|
| `aggregation_01_analysis.R` | Linear mixed models (LMM) and Aligned Rank Transform (ART) ANOVAs for aggregation and movement metrics; Spearman correlations for conspecific taxis and aggregation strength |
| `aggregation_02_figures.R` | Boxplots and correlation figures for aggregation/movement metrics |
| `valve_gape_01_analysis.R` | Beta-distributed GLMMs (`glmmTMB`) for valve gape (Models A & B); covariate checks (shell length, infection intensity) |
| `valve_gape_02_CV_analysis.R` | LMMs for log-transformed valve gape CV (Models A & B); covariate checks |
| `heart_rate_01_analysis.R` | LMMs for heart rate (Models A & B); covariate checks |

## How to run

1. Clone this repository and open it as an RStudio Project (so that working directory paths resolve correctly — all scripts use **relative paths** from the repository root).
2. Run scripts in `scripts/processing/` first, in the order listed above, to regenerate all processed datasets from raw data.
3. Run scripts in `scripts/analysis/` to fit the statistical models and reproduce all figures (saved to `outputs/figures/`).
